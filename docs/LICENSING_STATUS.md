# Licensing status

VirtualAuto does not yet declare a repository-wide open-source licence. Until
the owner selects and commits one, default copyright applies and public access
must not be interpreted as permission to copy, modify, or redistribute the
repository.

External tools retain their upstream licences. Referenced papers, OEM material,
community research, game data, supplied models, textures, and extracted assets
remain governed by their respective rights holders and are not relicensed by
VirtualAuto.

Before publishing reusable VirtualAuto code or accepting outside contributions,
the repository owner should select an appropriate code/documentation licensing
policy and decide whether research data requires separate terms.
