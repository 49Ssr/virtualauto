# Asset archaeology

Records for extracting, cataloguing, and interpreting production assets while
preserving provenance and unknown data.

## Active domains

- [DriveClub asset archaeology](driveclub/README.md) — package-to-resource
  boundaries, Evolution resource graphs, mesh/material research, parser failure
  controls, and the proposed F40 first target.

## Boundary

Raw game data and converted copyrighted assets are excluded from this
repository. Store only lawful tooling, schemas, manifests, observations, small
original evidence, and original reconstruction work.

An exported model is a derived source with a declared loss boundary. It is not
assumed to represent every stream, material parameter, hierarchy edge, or
runtime system used by the original engine.
