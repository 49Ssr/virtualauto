# Research

This directory holds reviewed knowledge records. The complete source remains
authoritative when generated retrieval views are introduced later.

## Domains

- [`automotive_materials/`](automotive_materials/README.md) — paint, composites,
  metals, glazing, polymers, and related body materials.
- [`geometry/`](geometry/README.md) — mesh rehabilitation, topology, normals,
  tolerances, and surface validation.
- [`rendering/`](rendering/README.md) — lighting, cameras, sampling, colour, and
  render validation.
- [`asset_archaeology/`](asset_archaeology/README.md) — source formats,
  extraction provenance, unknown fields, and semantic recovery.
- [`blender_workflows/`](blender_workflows/README.md) — version-specific Blender
  implementation knowledge.
- [`projects/`](projects/README.md) — target-specific status and changelogs.
- [`indexes/`](indexes/README.md) — generated navigation views.

Research claims must follow the [project doctrine](../lab/governance/PROJECT_DOCTRINE.md).
